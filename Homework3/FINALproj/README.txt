
Αυτή είναι η τελική μου εργασία. Παρακαλώ να διορθωθεί αντί της προηγούμενής μου
υποβολής.

Έχουν υλοποιηθεί όλοι οι overiding visitors και το parse όσων αρχείων java εκτέλεσα,
πραγματοποιείται με επιτυχία. Διορθώθηκαν τα λογικά λάθη της τελευταίς υλοποίησης και
υλοποιώ σωστά περισσότερα παραδείγματα.

Στη Makefile υπάρχουν οι εντολές για compile, clean και run. Τα αρχεία ll παράγονται
σε ένα φάκελο llFiles, ο οποίος πρέπει να υπάρχει από πριν στο directory. Επίσης οι
εντολές που έχω χρησιμοποιήσει για εγγραφές σε αρχείο προϋποθέτουν ότι το αρχείο δεν
προϋπήρχε. Αν υπάρχει ήδη, θα πάνε και θα συνεχίσουν να γράφουν σε αυτό από το σημείο
που είχε μείνει. Για αυτό το λόγο πριν από κάθε εκτέλεση καθάριζα και το φάκελο
llFiles.
